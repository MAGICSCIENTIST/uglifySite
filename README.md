# uglifySite
a library to min or copy your site's everything. split large project to small part.
这是一个能把你的站点能压缩的压缩,不能压缩的赋值出一个新project的类库. 主要用作自动的把庞大的项目分割成可用的小项目.
理论上不是网站也能用(→_→)
